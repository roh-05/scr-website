# Surrey Capital Research — Brand Guidelines

## Colour Palette

### Primary Colours

| Name     | Hex         | RGB           | Use                                    |
| -------- | ----------- | ------------- | -------------------------------------- |
| SCR Navy | `#3D5A80` | 61, 90, 128   | Headings, borders, primary text, frame |
| SCR Gold | `#B8963E` | 184, 150, 62  | Stag, accents, decorative elements     |
| White    | `#FFFFFF` | 255, 255, 255 | Backgrounds, reverse text              |

### Supporting Colours

| Name       | Hex         | RGB           | Use                                       |
| ---------- | ----------- | ------------- | ----------------------------------------- |
| Deep Navy  | `#263C55` | 38, 60, 85    | Dark backgrounds, heavy emphasis, footers |
| Pale Navy  | `#EBF0F5` | 235, 240, 245 | Section backgrounds, table headers        |
| Muted Gold | `#D4B86A` | 212, 184, 106 | Lighter accent, dividers                  |
| Dark Gold  | `#8A6E2A` | 138, 110, 42  | Gold on light backgrounds (contrast)      |
| Slate      | `#6B7F94` | 107, 127, 148 | Secondary text, captions, footnotes       |
| Off-White  | `#F7F8FA` | 247, 248, 250 | Page backgrounds, alternating table rows  |
| Charcoal   | `#1E2B36` | 30, 43, 54    | Body text on light backgrounds            |

### Colour Usage Notes

- **Navy + Gold** is the primary combination. Use gold sparingly — it is an accent, not a base colour.
- **Never place gold text on a white background** at small sizes; use Dark Gold (`#8A6E2A`) instead for contrast.
- **Pale Navy** works as a neutral background for tables and callout boxes.
- **Charcoal** (`#1E2B36`) rather than pure black for body text — softer and consistent with the navy family.

## Typography

No typeface is locked in via the logo (the logo uses a serif display font). Recommended pairings:

| Role             | Recommendation                                    | Notes                                    |
| ---------------- | ------------------------------------------------- | ---------------------------------------- |
| Display/headings | Garamond, EB Garamond, or Palatino                | Consistent with the serif logo lettering |
| Body text        | Georgia, or Merriweather (used in LaTeX template) | Readable at small sizes                  |
| Monospace/code   | Latin Modern Mono, or JetBrains Mono              | For code blocks and data tables          |

The LaTeX report template (`apcr-contemporary.cls`) uses Merriweather for body and Euler for mathematics — retain these.
